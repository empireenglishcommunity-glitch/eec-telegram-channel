# Channel Username Options

> **Decision needed:** Choose the @username for the Telegram announcement channel.
> **Criteria:** Short, memorable, available, brand-aligned, easy to type in Arabic keyboard.

---

| # | Username | Pros | Cons |
|---|----------|------|------|

| **3** | `@EEC_Official` | Short, clear it's official, professional | Underscore is harder to type on mobile |
| **4** | `@EmpireEnglish` | Clean, exact brand name | Might already be taken |
| **5** | `@macal_english` | Ties to MACAL personal brand | Mixed branding (MACAL vs Empire) |
| **6** | `@EECclub` | Very short, memorable | Less descriptive |

---

## My Recommendation: `@EmpireEnglishClub`

**Why:**
- Full brand name — zero ambiguity about what this is
- "Club" adds community feel (vs "Academy" or "School" which sound corporate)
- Matches the brand: "Empire English Club" (EEC)
- Searchable — someone typing "Empire English" in Telegram will find it
- Professional enough for long-term (won't need to change later)
- No underscore — easy to type and share verbally

**Backup if taken:** `@EmpireEngClub` or `@EEC_Channel`

---

## What to Check (You Do This)

1. Open Telegram → Search → type `@EmpireEnglishClub`
2. If it says "no results" → it's available → claim it in Channel Settings → Edit → Username
3. If taken → try `@EmpireEngClub` → then `@EEC_Official`

---

## Username Rules (Telegram)
- 5-32 characters
- Only letters, numbers, underscores
- Must be unique globally
- Can be changed later (but don't — consistency builds SEO/recognition)
