# 03 — Brand Voice & Writing System

> **Document:** How every post should SOUND. Tone, language, templates, and anti-patterns.

---

## 1. The MACAL Voice

Empire English is a sub-brand of MACAL Empire. The voice is:

| Dimension | Do This | Never This |
|-----------|---------|-----------|
| **Authority** | Speak from experience. State facts. Be direct. | Hedge, apologize, say "maybe" or "I think" |
| **Sarcasm** | Light, surgical jabs at bad methods. Scalpel, not sledgehammer. | Mean-spirited, mocking students, putting people down |
| **Paternal** | Protective. "I won't let you waste your time." Firm but caring. | Cold, corporate, robotic, distant |
| **Egyptian Arabic** | Natural masri dialect. Write how you talk. | Fusha (formal), awkward mixing, translated-sounding |
| **Confident** | "This is how it works." No disclaimers. | "I hope this helps" / "In my humble opinion" |
| **Exclusive** | "This isn't for everyone." Create belonging. | Desperate, begging, "please join us" |
| **Concise** | Say it in fewer words. Then cut more. | Rambling, over-explaining, padding |

## 2. Language Rules

| Rule | Example |
|------|---------|
| Primary language is **Egyptian Arabic** | "عارف ليه بتقول 'water' غلط؟" not "هل تعلم لماذا تنطق..." |
| English words stay in English | "الـ stress" not "النبرة" / "shadowing" not "التظليل" |
| Technical terms get 1-line Arabic explanation max | "IRT (نظام بيغيّر صعوبة الامتحان بناءً على مستواك)" |
| No formal Arabic EVER | Never use فصحى in posts. It creates distance. |
| Contractions OK in English examples | "don't" not "do not" when showing how Americans speak |
| Numbers in Arabic numerals | ٣، ٥، ٧ — except when mixed with English ("Level 3") |

## 3. Post Structure Rules

### Every Post Has:

1. **Hook (Line 1)** — Stops the scroll. Bold claim, question, or surprising fact.
2. **Body (Lines 2-8)** — Delivers the value. One idea, clearly.
3. **Takeaway (Last line)** — One sentence summary or breadcrumb.

### Formatting Rules:

- **Short paragraphs** — max 2-3 lines per paragraph. White space is your friend.
- **Emojis are functional** — use them as visual markers (🎯, 💣, 🔥), never decoration
- **Bold sparingly** — Telegram supports **bold**, use it for 1-2 key phrases per post
- **No hashtags** — this isn't Instagram. Hashtags in Telegram look amateur.
- **No "follow for more"** — the content speaks for itself or it doesn't
- **Signature:** End with nothing, or `ℳ` for brand story posts only

## 4. Post Templates

### Template A — Accent Lesson (🎯 AL)

```
[Hook — the problem/mistake in 1 line, Arabic]

🎯 [Sound/Pattern Name in English]

[2-3 lines: what Egyptians do wrong + why]

🎧 [Voice note attached]

[1-line takeaway — the fix]

—
[Optional breadcrumb: "ده واحد من ٤٤ صوت بنشتغل عليهم جوه Empire"]
```

**Example:**
```
كلمة "water" — لو بتنطقها "ووتر" أنت بتتكلم بريطاني من غير ما تعرف.

🎯 The Flap T

الأمريكان مش بيقولوا /t/ واضح في نص الكلمة. بيقولوا حاجة بين الـ d والـ t.
water = "wader"
better = "bedder"
little = "liddle"

🎧 [اسمع الفرق — الغلط الأول والصح]

الخلاصة: لو عايز تبان أمريكي — خلّي الـ T في نص الكلمة soft.

—
ده واحد من ٤٤ صوت بنشتغل عليهم جوه Empire. الـ pinned message فيه كل التفاصيل.
```

## 5. Anti-Patterns (NEVER Do This)

| Anti-Pattern | Why It's Bad |
|-------------|-------------|
| "صباح الخير 🌸 انهارده هنتكلم عن..." | Wastes the hook line on nothing |
| Long introductions before the point | People leave before reaching the value |
| "تابعونا للمزيد" / "شير لو عجبك" | Begging. The content should make them want to share. |
| Posting screenshots of other people's content | Shows you have nothing original to say |
| Multiple CTAs in one post | Confuses. One post = one purpose. |
| Posting in formal Arabic (فصحى) | Creates distance. You're not a textbook. |
| "إن شاء الله هنعمل..." (promises without delivery) | Erodes trust every time you don't deliver |
| Mixing English and Arabic mid-sentence randomly | Looks confused. Be intentional about code-switching. |
| Walls of text with no spacing | Unreadable on mobile. No one will finish it. |
| Posting at random times | Destroys the rhythm. Rhythm = trust. |

## 6. Voice Note Guidelines

Voice notes are your PRODUCT DEMO. They prove you can teach pronunciation.

### Rules:
- **Duration:** 30-90 seconds. Never longer.
- **Structure:** Wrong way (2-3 sec) → Explanation (10 sec) → Right way (5 sec) → Practice prompt (5 sec)
- **Energy:** Confident, clear, slightly intense. Not lazy or rushed.
- **Background:** Silent. No echo, no noise, no music.
- **Quality:** Clear mic. If your phone mic is bad, invest in a clip-on ($10).
- **Accent:** Demonstrate American English clearly. This is your proof.

### Voice Note Script Template:
```
[Say the word/phrase WRONG — how Egyptians say it]
"Most Egyptians say it like this: [wrong]"

[Explain the difference — 1-2 sentences]
"But Americans actually do [X] with their tongue/mouth/air"

[Say it RIGHT — slowly, then at natural speed]
"The correct way: [slow]... and at normal speed: [natural]"

[Practice prompt]
"Try it now. Say it 5 times. Record yourself."
```

## 7. The Breadcrumb System (Silent Funnel)

Every non-CTA post ends with a soft breadcrumb. These are NOT hard sells. They're reminders that there's more.

### Breadcrumb Examples (rotate, never repeat the same one twice in a row):

- "ده واحد من ٤٤ صوت بنشتغل عليهم جوه Empire."
- "الـ pinned message فيه كل التفاصيل."
- "عايز تعرف مستواك الحقيقي؟ الامتحان في أول رسالة."
- "جوه Empire بنعمل ده كل يوم. ٧ مهام. نظام. مش random."
- "ده اللي نظام بيعمله — مش course عادي."
- "الناس جوه عملوا كده في ٣٠ يوم. أنت ممكن كمان."

### Rules:
- Never more than 1 breadcrumb per post
- Always separated by `—` or empty line
- Never the focus — always an afterthought
- Skip it entirely on Social Proof posts (the proof IS the CTA)
